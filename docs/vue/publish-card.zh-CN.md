## 发布卡片

### 发布您的开发卡片
* 第一步:构建卡片
* 第二步:上传至业务平台资源托管服务器
* 第三步:通知管理员审核
* 第四步:审核通过后则上架成功

#### 构建卡片

开发人员在卡片工程中开发完成,并确保`package.json`中需要配置的配置项填写正确后,在工程中运行
```ssh
    npm run pub
```
开发人员将会在工程根目录得到一个根据`package.json`中的`name`字段构建好的`zip`包,将此zip包交于项目管理人员或者就是你自己,登入到`江西省水利事务业务平台`进行上传
